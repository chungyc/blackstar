This folder currently contains

the PPM star catalog mirrored from
http://tdc-www.harvard.edu/catalogs/ppm.html
in the file ppm.tar.gz

the binary header format
http://tdc-www.harvard.edu/catalogs/ppm.header.html
in the file header-format.txt

the binary entry format
http://tdc-www.harvard.edu/catalogs/ppm.entry.html
in the file entry-format.txt

Complete reference of the catalog:
Roeser, S., Bastian, U., 1991, Astronomisches Rechen-Institut, Heidelberg, Vols. I and II, printed by Spektrum Akademischer Verlag, Heidelberg
